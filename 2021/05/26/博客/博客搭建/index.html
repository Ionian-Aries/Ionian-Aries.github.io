*5.git 版本控制

解决方法：
1、添加到本地仓库

git add .

2、添加提交描述

git commit -m ‘提交’

3、提交前先从远程仓库主分支中拉取请求

git pull origin main

4、把本地仓库代码提交

git push -u origin main
